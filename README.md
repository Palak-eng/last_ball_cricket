# 🏏 Last Ball Cricket 

A fast, addictive, high-pressure cricket web game where every ball feels like a final-over thriller.

Built with **pure HTML, CSS, and JavaScript**, this game delivers dramatic gameplay, satisfying audio feedback, and replayable clutch scenarios.

---

## 🎮 Gameplay
SELECT THE MODE TO START

* Face intense scenarios like:

  * `6 needed off 1`
  * `12 off 4`
  * `Survive the hat-trick ball`
* Time your shot using a **precision timing meter**
* One tap decides everything:

  * 🟢 Perfect → SIX / SUPER SHOT
  * 🟡 Good → FOUR / RUN
  * 🔴 Bad → DOT / WICKET
